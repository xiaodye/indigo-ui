`Indigo UI` 是一款由 React + TypeScript 开发的组件库 🎉。参考了 Antd Design。

作为一个组件库来说，它并不完美，一切也是为了学习而生 🖊，在此也不推荐投入生产环境 ⚠

当然，也可以以其中部分源码（or 思路）为地基，从而完善成自己的作品 🏢

## 安装

从 yarn、npm 或者 pnpm 安装并引入 indigo-ui

```bash
pnpm add @indigo-ui/components
```
